\doxysubsection{Detailed Description}
ESP32 BLE Central with RFID and LCD UI. 

This sketch implements an ESP32 BLE central device that\+:
\begin{DoxyItemize}
\item Scans and connects to a BLE peripheral (with Button + IMU characteristics)
\item Estimates distance from RSSI using a path-\/loss model
\item Displays IMU movement state and distance on an I2C LCD
\item Uses Free\+RTOS tasks for concurrency (scanner, distance calc, UI, button, RFID, reset)
\item Integrates RC522 RFID for access control, requiring authorized UID
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Uses ESP32 Arduino Core 3.\+x (Nim\+BLE backend). 
\end{DoxyNote}
