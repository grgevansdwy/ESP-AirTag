void schedulerTask ( void * pvParameters)

Implements Shortest Remaining Time (SRT) scheduling algorithm.

◆ setup()

void setup ( )

Initializes serial communication, LCD display, and creates all FreeRTOS tasks with custom scheduler.

This function sets up all tasks (LED, counter, alphabet) in a suspended state initially, and starts the scheduler task with the highest priority.
